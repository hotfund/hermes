## 기술 메모

- `computer_use` 도구가 세션에 로드되지 않아 **cua-driver CLI 직접 호출** 방식 사용
- `cua-driver.exe call get_desktop_state` → base64 PNG 추출 → ffmpeg crop
- `cua-driver.exe call scroll` (foreground 모드) → 페이지별 캡처
- `cua-driver.exe call bring_to_front` → Hermes 창 가림 문제 해결
- 총 28페이지 스크롤 캡처, vision_analyze로 820/888 판별
