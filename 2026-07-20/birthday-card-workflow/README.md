## 2. 생일축하카드 자동생성 워크플로우

### 발송 스케줄
- 생일 **당일 오전 7시** 실행
- 교수님이 매일 6:50에 PC 켬 → 켜지면 즉시 실행
- 최종 "보내기" 버튼은 교수님이 직접 누름

### 실행 순서 (더깨비 + cua-driver CLI)

1. 더깨비 크롬 bring_to_front
2. "영상/카드 만들기" 탭 → "이미지·카드" 선택
3. **"어떤주제?"** → **"🎂 생일 축하"** 클릭
4. **"분위기"** → **"고객이름 3행시 + 건배사"** 클릭
5. **"받는 사람 (고객) 이름"** 입력란에 이름 입력 (클립보드 Ctrl+V)
6. **"🎉 이름넣고 건배사 3행시 짓기"** 버튼 클릭
7. **8초 대기** → 3행시 결과 생성
8. **"메시지 칸에 넣기"** 버튼 클릭
9. 아래로 스크롤 → **"🖼 이미지 만들기"** 버튼 클릭
10. **12초 대기** → 카드 이미지 생성
11. **"카카오톡"** 버튼 클릭 → 링크 클립보드 복사
12. 카카오톡 전환 → 친구 검색 (Ctrl+Shift+F) → 이름 검색
13. 채팅창 열기 → Ctrl+V 링크 붙여넣기
14. **최종 보내기는 교수님이 직접**

## 3. Pitfalls (실패에서 배운 것)

1. **카카오톡 "생일인 친구" 팝업에 날짜 구분이 있다** — "오늘 생일", "다가오는 생일 7월 N일" 헤더가 중간중간 나옴. 날짜를 추측하면 안 되고, 반드시 스크롤하면서 헤더 확인.
2. **이전 DB(14명)는 날짜를 지어낸 것** — 팝업 구조를 이해 못하고 임의로 날짜를 배정. 완전 폐기.
3. **"메시지 칸에 넣기" 클릭이 안 먹힘** — 더깨비 버그. 교수님이 수정 예정.
4. **카카오톡 포그라운드 전환 실패** — cua-driver-uia worker 미설치. Alt+Tab → bring_to_front 순서로 우회.
5. **Ctrl+V가 안 먹힐 때** → 입력란 double_click → Ctrl+A → Ctrl+V → Tab
6. **더깨비 "카카오톡" 버튼** → 링크가 클립보드에 복사됨. 카톡 검색어로 덮어쓸 수 있으니 순서 주의.
7. **820/888 아닌 사람 올리지 말 것** — 시킨 것만 정확히.
8. **데이터를 지어내지 말 것** — 확인 안 된 건 "모른다"고 말할 것.
