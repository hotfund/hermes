## 🎯 경쟁 포지셔닝

### 더깨비만의 포지션: "감성적 관계 자동화"

```
경쟁사 전부: 쿠폰 → 할인 → 재방문 유도 (실리적)

더깨비:      안부 → 축하 → 감동 → 자연스러운 재방문 (감성적)
             ↑
         이 포지션을 가진 경쟁사 = 0개
```

### 핵심 메시지
- **보험설계사 시장**: "더깨비는 AI가 고객 안부를 챙겨서, 설계사가 영업에만 집중하게 합니다"
- **소상공인 시장**: "쿠폰 말고 마음을 보내세요. AI가 단골 한 명 한 명을 기억합니다"

### vs 경쟁사 원라이너

| 경쟁사 | 더깨비가 이기는 이유 |
|--------|-------------------|
| 보케어 | AI 개인화 카드 + 금소법 자동 준수 = 보케어에 없는 기능 |
| 모아톡 | 카톡 발송은 모아톡, 그 위의 AI 두뇌는 더깨비 |
| 도도포인트 | 적립 포인트 → 충성도 낮음. 안부 카드 → 충성도 높음 |
| 토스플레이스 | 무료지만 감성 터치 0. "토스가 못하는 것 = 마음 전하기" |
| 네이버 | 플랫폼은 무료, 관계의 깊이는 유료 |
