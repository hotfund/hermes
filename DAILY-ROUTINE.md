# Hermes 매일 할 일 (더깨비 생일 관리)

## 매일 아침 루틴 (교수님 PC 켜면 즉시)

### 작업 1: 오늘 생일자 카드 보내기
1. 카카오톡 "생일인 친구" 열기
2. "오늘 생일" 중 820/888 있는지 확인
3. 있으면 → theggabi.com에서 생일축하카드 생성
   - 어떤주제: 생일축하카드
   - 분위기: 고객이름 3행시 + 건배사
   - 이름 입력 → 3행시짓기 → 메시지칸에넣기 → 이미지만들기 → 카카오톡
4. 카카오톡 채팅창에 링크 띄워놓기
5. 교수님이 "보내기" 누름

### 작업 2: 7일 후 생일자 theggabi 등록
1. 카카오톡 "생일인 친구" 열기
2. "다가오는 생일"에서 7일 후 날짜까지 스크롤
3. 해당 날짜에 820/888 포함자 찾기
4. 있으면 → theggabi.com 고객DB에 등록

### 작업 후
- 결과를 theggabi-research git에 기록 + push (ik@incutek.net)
